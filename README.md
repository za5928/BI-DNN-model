**Repository Overview**

This repository contains the official implementation and synthetic data experiment for the paper: "***A Behaviorally-Informed Deep Neural Network for Latent Variable Choice Modeling***".

File Structure:

01_Data_Generation.ipynb: Generates the synthetic MaaS dataset.

02_Biogeme_Estimation.ipynb: Traditional ICLV model estimation.

03_BIDNN_Training.ipynb: Training the proposed BI-DNN model.

04_FDNN_Benchmark.ipynb: Training the F-DNN.

05_Behavioral_Analysis_and_Interpretability.ipynb: Model comparison and point elasticities.

Pre-trained weights (.pth) and Biogeme estimation results (.pickle) are provided for one-click reproducibility without re-training.
